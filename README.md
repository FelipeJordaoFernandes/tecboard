# 🚀 Tecboard

Landing page desenvolvida para apresentar o **Tecboard**, uma solução de monitoramento de sistemas com foco em acompanhamento em tempo real.

---

📖 Sobre o projeto

O **Tecboard** é uma página de apresentação (landing page) que simula um produto voltado para monitoramento de aplicações e sistemas.

A ideia central do projeto é transmitir, de forma visual e objetiva, a importância de acompanhar métricas e eventos em tempo real, utilizando um design moderno e responsivo.

> 🟣 *"Fique de olho no que importa"*
